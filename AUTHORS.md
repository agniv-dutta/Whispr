## Primary Author

**Agniv Dutta** — Initial work and ongoing development

## Project Context

Whispr was built as a **Scaler SDE Fullstack Assignment** demonstrating
proficiency in:

- React/Next.js frontend development
- FastAPI/Python backend development
- Real-time communication with WebSockets
- Database design with SQLAlchemy
- Deployment and DevOps (Vercel, Render, GitHub Actions)

## Design Inspiration

The user interface and user experience are heavily inspired by
**[Signal Messenger](https://signal.org/)**, specifically:

- Conversation list layout and message bubble styling
- Dark/light theme color palettes
- Typing indicator and presence patterns
- Settings menu navigation structure
- Media attachment workflow

## Technologies

This project would not be possible without these amazing technologies:

- **[Next.js](https://nextjs.org/)** by Vercel — React framework
- **[FastAPI](https://fastapi.tiangolo.com/)** — Python web framework
- **[Tailwind CSS](https://tailwindcss.com/)** — Utility-first styling
- **[Radix UI](https://www.radix-ui.com/)** — Accessible headless components
- **[Zustand](https://github.com/pmndrs/zustand)** — State management
- **[Framer Motion](https://www.framer.com/motion/)** — Animation library
- **[Recharts](https://recharts.org/)** — Charting library
- **[Sonner](https://sonner.emilkowal.ski/)** — Toast notifications
- **[Lucide](https://lucide.dev/)** — Icon set
- **[SQLAlchemy](https://www.sqlalchemy.org/)** — Python ORM
- **[Alembic](https://alembic.sqlalchemy.org/)** — Database migrations
